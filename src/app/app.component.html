Analog Clock

Simple analog clock. Currently only size is configurable in pixels. Supports configuration for timezone and color scheme.

<ng-pluggable-analog-clock [size]="90" [gradiantA]="'#FFFF00'" [gradiantB]="'#00AA00'" [timezoneOffsetInMinutes]="540"></ng-pluggable-analog-clock>
Microphone

Click the microphone, hold it and speak to record your voice. It emits an event "recognize" which contains the audio blob. You can pass that audio to a speech recognition API. You can use online speech recognition services by Google, Microsoft, Amazon, etc. or create your own STT API like me.

<ng-pluggable-microphone (recognize)="processAudioBlob($event)">
    <button class="btn btn-secondary microphone">
        <fa name="microphone"></fa>
    </button>
</ng-pluggable-microphone>
