Number Internationalization

Hindi:
Kannada:

Digit by digit translation of number from English to other language. RTL languages and languages which do not follow digit by digit logic throughout (e.g. Japanese), are not supported yet.
By default, the component supports a few locales which you can test by passing locale name in [selectedLocale] input and the number to be translated in [value] input.
You can also pass the locales data by yourself as part of optional [locales] input.

<ng-pluggable-number-i18n [locales]="locales">
[value]="value">
[selectedLocale]="'hi-IN'">
</ng-pluggable-number-i18n>
Currency Converter

Select any two currencies, to start with, and quickly check conversion between them. You can also keep adding more currencies and get all the converted values at once.
By default, the component fetches currency coversion rates from this url. You can also pass the currency data by yourself as part of optional [config] input.

<ng-pluggable-currency-converter [config]="config">
</ng-pluggable-currency-converter>
Microphone

Click the microphone, hold it and speak to record your voice. It emits an event "recognize" which contains the audio blob. You can pass that audio to a speech recognition API. You can use online speech recognition services by Google, Microsoft, Amazon, etc. or create your own speech-to-text API.

<ng-pluggable-microphone (recognize)="processAudioBlob($event)">
    <button class="btn btn-secondary microphone">
        <fa name="microphone"></fa>
    </button>
</ng-pluggable-microphone>
